# WTbrainwave Ai

## Project Overview
WTbrainwave Ai is a multi-agent SaaS platform enabling users to manage subscriptions, interact with AI agents for code generation, correction, and multimedia creation. Users can pay via Visa for weekly, monthly, or yearly plans. The platform is built using Next.js, React, TypeScript, and hosted online.

## Features
- Multi-agent AI system for code, image, video, and music generation
- User authentication and subscription management
- Integration with multiple APIs
- Chrome, Edge, and Opera extension for automating browser tasks
- Responsive and modern frontend design

## Technologies Used
- **Frontend:** React, TypeScript, Next.js
- **Backend:** Next.js API routes / serverless
- **Database:** Your choice (MySQL.)
- **Payment:** Visa / Stripe integration
- **Other:** Browser extension support (Chrome, Edge, Opera)

## Installation
1. Clone the repository:
```bash
git clone https://github.com/Wissemtriki/WTbrainwave-Ai.git
cd WTbrainwave-Ai
2.Install dependencies:
npm install
# or
yarn install
3.Add your .env variables (API keys, database connection, payment configs)

4.Run the development server:
npm run dev
# or
yarn dev
